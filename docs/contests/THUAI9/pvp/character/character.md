# 角色（PVP）

三种角色类型，由 Team 在己方工厂消耗算力召唤。每队最多 3 名角色。

## 角色数值表

| 属性 | Drone（无人机） | Robot（机器人） | AutonomousCar（自动驾驶汽车） |
|:----:|:----:|:----:|:----:|
| 血量 (HP) | 100 | 150 | 100 |
| 攻击力 (ATK Power) | 40 | 30 | 18 |
| 攻击范围 (ATK Size) | 1000 | 1000 | 1000 |
| 防御力 (Robust) | 10 | 15 | 8 |
| 视野范围 (ViewRange) | 7000 (7格) | 5000 (5格) | 5000 (5格) |
| 负载上限 (Load) | 5 | 5 | 5 |
| 效率 (Efficiency) | 1 | 1 | 2 |
| 移动速度 (Speed) | 5000 | 5000 | 5000 |
| 算力消耗 (Cost) | 50 | 50 | 50 |
| 半径 | 300 | 300 | 300 |

## 主动状态

角色同一时刻只能处于一种主动状态：

| 状态 | 说明 |
|------|------|
| `IDLE` | 空闲，可接受新指令 |
| `MOVING` | 移动中，新的 Move 指令会中断当前移动 |
| `HARVESTING` | 采集资源中 |
| `ATTACKING` | 攻击中，有攻击冷却 |
| `OCUPPYING` | 占领算力中心中 |
| `TRADING` | 交易中 |
| `KNOCKED_BACK` | 被击退（被动），强制中断主动动作 |
| `DECEASED` | 已死亡，从地图移除 |

## 伤害计算

```
实际伤害 = Max(攻击方ATK - 防御方Robust, 0)
```

- 攻击为**锁定**攻击，无法闪避
- 角色死亡后，队伍可重新花费算力召唤新角色

## 效率的影响

- **采集速度**：实际采集 = 80 × (1 + 效率等级 × 0.2) 单位/秒（每帧 `200/50 = 4` 单位）
- **占领速度**：实际占领时间 = 10000 / (1 + 效率等级 × 0.2) ms

## 视野

- 每个角色以自身为圆心、ViewRange 为半径观察敌方角色
- 障碍物（Barrier）阻挡视野
- 草丛（Bush）中的角色对外不可见，发起攻击后破隐
- 地图上所有静态物体（资源/市场/算力中心/工厂）的位置始终可见
