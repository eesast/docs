# 算力中心（PVP）

散布在地图各个点位，角色可进行**占领（Occupy）**为工厂提供算力加成。

## 属性

| 属性 | 值 |
|:----:|:--:|
| 半径 | 500 |
| 基础占领时间 | 10 秒（10000ms） |

## 占领条件

- **仅 Drone 和 Robot** 可执行占领
- 角色需处于**空闲状态（IDLE）**
- 角色需在算力中心格子的**九宫格内**

## 占领时间

```
实际时间 = 10000 / (1 + 角色效率 × 0.2) ms
```

例如：效率=2 的 AutonomousCar 无法占领；效率=1 的角色需 10s；升级 Efficiency +2 后效率=3，需 10000/(1+0.6) = 6.25s。

## 占领收益

- 每个已占领的算力中心为工厂提供 **+2 算力/秒**
- 多个算力中心加成**可叠加**

## 状态

| 状态 | 说明 |
|------|------|
| `OCCUPYABLE` | 可占领 |
| `OCCUPIED` | 已被某队伍占领 |
| `ROBBED` | 被抢夺（预留） |
