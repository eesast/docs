# 资源（PVP）

散布在地图各个点位，角色可进行**采集（Harvest）**。

## 属性

| 属性 | 值 |
|:----:|:--:|
| 血量 (HP) | 500 |
| 半径 | 500 |
| 基础采集速度 | 80/秒（4/帧） |

## 采集

- 条件：角色位于资源格子的**九宫格内**
- 速度：`80 × (1 + 角色效率 × 0.2)` 单位/秒（每帧 `200/50 = 4` 单位）
- 采集的资源（Source）存入己方工厂
- 资源血量归零后变为 `HARVESTED`，不可再采集
- 不同队伍的角色可同时采集同一资源

## 状态

| 状态 | 说明 |
|------|------|
| `HARVESTABLE` | 可采集 |
| `BEING_HARVESTED` | 采集中 |
| `HARVESTED` | 已耗尽 |
