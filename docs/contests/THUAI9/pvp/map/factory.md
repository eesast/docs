# 工厂（PVP）

每支队伍拥有一座工厂，位于地图四角之一，是队伍的**核心基地**。

## 工厂属性

| 属性 | 初始值 | 上限 |
|:----:|:------:|:----:|
| 血量 (HP) | 1000 | 3000 |
| 防御力 (Robust) | 20 | 50 |
| 仓储 (Storage) | 5 | 150 |
| 效率 (Efficiency) | 1 | 2 |
| 算力 (Computing Power) | 100 | — |
| 资源 (Source) | 0 | — |
| 半径 | 500 | — |

## 工厂功能

### 召唤角色
- 消耗 50 算力召唤 Drone / Robot / AutonomousCar
- 条件：`CanRecruit = true`，且算力充足
- 角色出生在工厂附近（自动避开碰撞）

### 生产产品
- 消耗 Source（资源）生产商品到工厂库存
- 条件：`CanProduce = true`，Source 充足，库存未满
- 生产可被 `Production` 科技加速，`Cost` 科技降低成本

### 升级科技
- 消耗算力升级，详见 [科技系统](../character/tech.md)

## 算力产出

- 基础：**1 点/秒**
- 每个已占领算力中心：**+2 点/秒**

## 被攻击

- 前 **20 秒**工厂处于**无敌状态**，不受任何伤害
- 受到攻击时，实际伤害 = `Max(ATK - Robust, 1)`
- 攻击方获得等同于工厂实际损失血量的分数
- 工厂被攻击后进入 **1s 停摆**（`CanProduce` 和 `CanRecruit` 均为 false）
- 工厂被摧毁（HP=0）：攻击方获得 **1000 分**，工厂所属队伍无法再生产/招募
