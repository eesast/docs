# 市场（PVP）

角色在市场进行商品**买卖（Trade）**。卖出商品获得分数，买入商品消耗分数。

## 市场类型

| 类型 | 价格倍率 |
|:----:|:--------:|
| 小型市场 (Small) | ×1.1 |
| 中型市场 (Medium) | ×1.3 |
| 大型市场 (Large) | ×1.5 |

## 交易条件

- 角色需在市场格子的**九宫格内**
- **卖出**：角色背包有对应商品 → 获得 `价格 × 数量` 分（受 Price 科技加成）
- **买入**：队伍分数充足 → 消耗 `价格 × 数量` 分，商品存入角色背包

## 价格衰减

每类商品独立计算衰减。当某类商品在市场累计交易量增加时，价格会逐步下降：

```
价格 = 基础价格 × 市场倍率 × max(1.0 - floor(累计交易量 / 50) × 0.05, 0.5)
```

| 参数 | 值 |
|:----:|:--:|
| 衰减阈值 | 每 50 单位交易量 |
| 衰减步长 | -0.05 |
| 最低倍率 | 0.5（价格不会低于基础价的一半） |

## 市场事件

服务器每隔 **30 秒**可能触发**市场事件**（Market Event），临时改变某些商品的市场价格。
